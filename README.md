# Image Viewer
Simple image viewer written in Qt.

## Dependencies
##### Ubuntu packages
```
cmake
mesa-common-dev
```

## Clone & Compile
##### 1. Clone
```shell
git clone https://github.com/hladik-dan/ImageViewer.git
```

##### 2. Compile
```shell
cd ImageViewer
cd Build
cmake ..
make
```

## Run | Install
##### Run
```shell
make run
```

##### Install
```shell
sudo make install
```
